# LLM Rules Set

A collection of system rules and protocols for optimizing Large Language Model (LLM) interactions.

## What This Project Does

This repository contains structured rule sets that define how LLMs should communicate, plan, and execute tasks. The rules are designed to ensure consistent, efficient, and reliable AI-assisted workflows.

## Current Rules

- **CONCISE & DIRECT COMMUNICATION** - Protocols for streamlined, no-fluff responses
- **DECOUPLED PLAN-AND-EXECUTE** - Two-phase workflow system for systematic task handling

## Contributing

When adding new rules:

1. **Follow existing patterns** - Use the same format and structure as current rules
2. **Keep it focused** - Each rule should address one specific aspect of LLM interaction
3. **Test thoroughly** - Ensure new rules don't conflict with existing ones
4. **Document clearly** - Explain the purpose and provide examples

### File Naming Convention

```
# SYSTEM_RULE: [RULE_NAME].md
```

### Rule Structure

- Clear title and purpose
- Numbered sections for different aspects
- Bullet points for specific guidelines
- Practical examples where helpful
